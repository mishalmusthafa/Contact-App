# Contact MERN App
This is a MERN app project for storing the Contacts the people

## Usage
Rename the .envexample to .env and add your MONGO_URI

### Install Dependencies

```
# Backend Dependencies
npm install

# Frontend Dependencies
cd frontend 
npm install
```

### Run server
```
npm run server
```
